# Movie & Series Catalogue

A simple full-stack application for browsing movies and series, with user authentication and favourites.

## Features

- Browse movies and series
- Search, filter by genre/type, and sort by title or year
- User registration and login (JWT authentication)
- Add/remove favourites per user

## Tech Stack

- **Frontend:** Vue 3 + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Authentication:** JWT + bcrypt
- **Data Storage:** JSON files (for simplicity)
- **CORS** enabled for frontend-backend communication
